## 统计学分析: 好机会 vs 坏机会

### BEARISH_SINGULARITY (271个)

**分组**:
- 好机会: 138个，平均盈亏 6.57%
- 坏机会: 133个，平均盈亏 1.79%

**显著差异特征** (p<0.05 或 Cohen's d>0.5):

| 阶段 | 特征 | 好机会均值 | 坏机会均值 | Cohen's d | 显著性 |
|------|------|-----------|-----------|-----------|--------|
| 黄金平仓 | 张力 | -0.3782 | 0.2141 | -1.030 | *** 超大 |
| 黄金平仓 | 加速度 | 0.0324 | -0.0183 | 0.723 | *** 中等 |
| 黄金平仓 | 量能 | 1.5452 | 1.1147 | 0.592 | *** 中等 |

**关键洞察**:
- **黄金平仓时刻的张力**是最强预测因子 (d=-1.030, ***)
  - 好机会: 平仓时张力为负（-0.3782）
  - 坏机会: 平仓时张力为正（0.2141）
  - **结论**: 在张力转负时平仓更好！

- **黄金平仓时刻的加速度**也有预测力 (d=0.723, ***)
  - 好机会: 加速度为正（0.0324）
  - 坏机会: 加速度为负（-0.0183）

- **黄金平仓时刻的量能** (d=0.592, ***)
  - 好机会: 量能更高（1.55 vs 1.11）

**相关性分析**:
- 信号前5周期平均张力与盈亏**负相关** (-0.175)
- 黄金开仓等待周期与盈亏**负相关** (-0.286)
  - **结论**: 等待时间过长反而降低盈亏

## V7.0.5 优化建议

### 1. 动态等待策略

**当前策略**: 信号时刻立即开仓
**问题**: 错失平均1.66-1.79%的价格优势

**建议**:
```
IF 信号类型 == BEARISH_SINGULARITY:
    IF 信号时刻张力 > 0.8:
        等待3-5周期
    ELSE:
        等待0-2周期

IF 信号类型 == BULLISH_SINGULARITY:
    IF 信号时刻张力 < 0.8:
        等待5-8周期
    ELSE:
        等待0-3周期
```

### 2. 智能平仓策略

**当前策略**: 固定止损/止盈
**问题**: 未考虑平仓时刻的张力变化

**建议**:
```
IF 方向 == SHORT:
    IF 当前张力 < -0.3 AND 当前加速度 > 0 AND 当前量能 > 1.2:
        立即平仓
    ELSE:
        继续持有

IF 方向 == LONG:
    IF 当前张力 > 0.3 AND 当前加速度 < 0 AND 当前量能 > 1.2:
        立即平仓
    ELSE:
        继续持有
```

### 3. 信号前特征过滤

**BEARISH_SINGULARITY**:
- 如果信号前5周期平均张力过高（>0.9），降低仓位
- 如果等待周期预计>8，考虑跳过

**BULLISH_SINGULARITY**:
- 如果信号前5周期平均张力过低（<0.7），降低仓位
- 如果黄金开仓量能过高（>1.4），降低仓位
