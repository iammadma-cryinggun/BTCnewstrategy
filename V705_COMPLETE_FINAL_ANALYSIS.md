## 核心发现总结

### 1. 震荡信号（主要信号）

#### LOW_OSCILLATION（低位震荡做多）- 201个

**盈亏表现**：
- 好机会（101个）：平均盈亏 **5.49%**
- 坏机会（100个）：平均盈亏 2.00%

**关键特征**：
- ⭐⭐⭐ **平仓时量能比率**：Cohen's d=0.71（大效应，p<0.0001）
  - 好机会：1.77，坏机会：1.23
  - Youden最优阈值：≥1.36（好机会率66.1%）

**最优策略**：
```
开仓：信号出现即可
平仓：量能比率 ≥ 1.36 AND 盈亏 > 2% → 平仓（好机会率66.1%）
```

#### HIGH_OSCILLATION（高位震荡做空）- 128个

**盈亏表现**：
- 好机会（65个）：平均盈亏 **4.68%**
- 坏机会（63个）：平均盈亏 **-78.40%**（风险极大！）

**关键特征**：
- ⭐ **信号前张力**：p=0.0332（显著）
  - 好机会：1.01，坏机会：0.77
- ⭐ **信号前量能比率**：p=0.0003（极显著）
  - 好机会：0.92，坏机会：1.14（好机会量能更低！）
- ⭐⭐ **开仓时量能比率**：Cohen's d=-0.52（中等效应，p=0.0055）
  - 好机会：1.14，坏机会：1.47（好机会量能更低）
- ⭐⭐ **平仓时量能比率**：Cohen's d=0.79（大效应，p<0.0001）
  - 好机会：1.82，坏机会：1.19
- ⭐⭐ **平仓时加速度**：Cohen's d=0.54（中等效应，p=0.0028）
  - 好机会：0.0106，坏机会：0.0027

**最优策略**：
```
开仓：信号出现且量能比率 < 1.3（量能不要太高）
平仓：量能比率 ≥ 1.82 AND 盈亏 > 2% → 平仓（好机会率增加）
      OR 加速度 ≥ 0.0017 AND 盈亏 > 2% → 平仓（好机会率69.6%）
```

```python
if signal_type == 'LOW_OSCILLATION':

        close_position()
```

#### HIGH_OSCILLATION（做空）
```python
if signal_type == 'HIGH_OSCILLATION':
    # 开仓：检查量能不要太高
    if volume_ratio < 1.3:
        entry_signal = True
    else:
        entry_signal = False  # 跳过高量能信号

    # 平仓：量能放大或加速度向上
    if volume_ratio >= 1.82 and pnl > 2:
        close_position()
    elif acceleration >= 0.0017 and pnl > 2:
        close_position()
```

```python
if signal_type == 'BEARISH_SINGULARITY':
```

### 1. 奇点信号极其罕见但质量高

- 阈值在1.3-1.8之间

### 4. DXY燃料对奇点信号很重要
