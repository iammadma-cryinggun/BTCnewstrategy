## 三、量化策略规则

### 3.1 开仓规则

#### SHORT开仓:
```python
# 首次信号触发条件
if 张力 > 0.5 and 加速度 < 0:
    等待_周期数 = 1  # 等待4小时

# 1个周期后检查
if 当前张力 > 0.47 and 当前加速度 < 0 and 量能比率 < 1.0:
    开空()
```

#### LONG开仓:
```python
# 首次信号触发条件
if 张力 < -0.5 and abs(加速度) < 0.0001:
    等待_周期数 = 1  # 等待4小时

# 1个周期后检查
if 当前张力 < -0.55 and 当前加速度 > 0:
    开多()
```

### 3.2 平仓规则

#### SHORT平仓:
```python
# 开仓后持续监控
张力下降比例 = (开仓时张力 - 当前张力) / 开仓时张力

if 张力下降比例 > 0.4 and (当前加速度 > 0 or abs(当前加速度) < 0.001):
    if 当前量能比率 > 1.0:
        平空()
```

#### LONG平仓:
```python
# 开仓后持续监控
张力变化比例 = (当前张力 - 开仓时张力) / abs(开仓时张力)

if 张力变化比例 > 1.0 and 当前张力 > 0 and 当前加速度 > 0:
    平多()
```

## 五、实战建议

1. **不要立即开仓**: 首次信号后等待1-2个周期(4-8小时)
2. **SHORT策略**: 等张力确认后>0.5，加速度保持负值，量能<1.0
3. **LONG策略**: 等张力<-0.55，加速度必须转正
4. **SHORT平仓**: 张力下降40%+，加速度转正，量能放大
5. **LONG平仓**: 张力从负转正，变化幅度>100%，保持正加速度
